# BHA Documentation

- tutorial: learn by doing (`quickstart.md`)
- how-to: solve specific operational tasks (`installation.md`, `export_ci.md`, `troubleshooting.md`)
- reference: exact command/API behavior (`cli_reference.md`, `lsp_reference.md`, `suggestions_reference.md`)
- explanation: internal design and tradeoffs (`architecture.md`, `docs_strategy.md`, `repo_validation.md`)
